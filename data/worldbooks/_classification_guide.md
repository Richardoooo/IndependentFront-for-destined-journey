# 世界书分类指南

## 分类定义

| 分类 | 说明 | 典型标志 |
|------|------|---------|
| **世界总览** | 世界观设定、纪元历史、政治社会、势力概况、文化习俗、魔法体系基础 | `[世界主设定]` `[政治与社会]` `[文化]` `概览` `势力-` |
| **地区** | 地理环境、城市/城镇/区域描述、建筑、气候、旅行规则 | `[地理]` `城镇-` `城市` `区域` `环境` `地块-` `冒险区域` |
| **角色** | 角色/NPC 传记、种族/血脉定义、登神长阶、技能装备规则 | `[角色]` `种族-` `角色-` `血脉` `DLC`角色 |
| **命定核心** | 命定之诗系统、命定之灵、各个命定核心（茶茶/梅林/莉莉丝等） | `命定系统` `命定之诗` `核心` |
| **数值** | 数值表、公式、计算规则、品质/DMG/HP 等可量化内容 | `[协议]` `[数值]` `骰子` `DC` `公式` `乘数` `伤害` |
| **叙事指导** | 告诉 AI 怎么叙事、格式要求、美化规则、输出规范 | `[指导]` `[美化]` `格式` `角色扮演指南` |
| **EJS相关** | 任何含 `<%` 或 `{{getvar::` 或 `{{setvar::` 的 EJS/模板条目 | 内容里有 `<%` `{{getvar` `{{setvar` |
| **事件相关** | DLC 事件链、长篇事件描述、事件触发逻辑 | `[事件]` `event` `事件标题` |
| **其他** | 空内容、纯标记（如 `➡️`）、看不出实际意义的条目 | 空 content、分隔符、占位符 |

## 优先规则

1. **先看 content 前 300 字** — comment 可能不准（老版本遗留）
2. **EJS 优先** — 只要 content 里有 `<%` 或 `{{getvar::` 就归 EJS
3. **数值 > 叙事** — 如果一条既讲数值公式又讲叙事规则，优先归数值
4. **事件 > 角色** — DLC 事件中出现的角色信息，如果主体是事件链，归事件相关
5. **角色 > 地区** — 如果一条主要讲某个角色但也提到地点，归角色

## 输出格式

```
UID ||| 分类 ||| 理由（简短）
```

示例：
```
127 ||| 叙事指导 ||| 隐蔽空间生成指南，教AI怎么描述隐蔽空间
583 ||| 地区 ||| 奥古斯提姆帝国地理
2117 ||| 角色 ||| 精灵种族扩展
5831 ||| EJS相关 ||| 含 <% EJS 模板 + getvar
7586 ||| 世界总览 ||| 翼民圣纹力量体系
```
